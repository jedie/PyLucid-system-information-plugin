System information

a PyLucid CMS Plugin

h2. setup

Quick how to install the plugin into pylucid:

pre. ~$ mkdir pylucid_plugins
~$ cd pylucid_plugins
~/pylucid_plugins$ git clone git://github.com/jedie/PyLucid-system-information-plugin.git
~/pylucid_plugins$ ln -s ~/pylucid_plugins/PyLucid-system-information-plugin/system_information ~/PyLucid_env/src/pylucid/pylucid_project/external_plugins/

If you don't have git, but suversion, you can checkout with this command:

pre. ~/pylucid_plugins$ svn checkout http://svn.github.com/jedie/PyLucid-system-information-plugin.git

how to update it:

pre. ~$ cd pylucid_plugins/PyLucid-system-information-plugin
~/pylucid_plugins/PyLucid-system-information-plugin$ ./update

the update script runs "git pull origin master" or "svn update"
